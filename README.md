# 🎬 Movie Genre Classifier

A Machine Learning project that predicts one or multiple movie genres based on plot summaries using Natural Language Processing (NLP) and supervised machine learning techniques. The model processes movie descriptions, extracts textual features, and predicts the most relevant genres.

---

## 📌 Overview

Movie genre classification is a multi-label text classification problem where a single movie can belong to multiple genres. This project demonstrates an end-to-end machine learning pipeline including data preprocessing, feature engineering, model training, prediction, and evaluation.

---

## 🚀 Features

- Multi-label movie genre prediction
- Text preprocessing and cleaning
- Feature extraction using TF-IDF
- Machine Learning model training
- Genre prediction from movie descriptions
- Performance evaluation using classification metrics
- Interactive Jupyter Notebook implementation

## ⚙️ Workflow

1. Load movie dataset
2. Clean and preprocess movie descriptions
3. Perform text tokenization and preprocessing
4. Convert text into TF-IDF feature vectors
5. Train multi-label classification model
6. Evaluate model performance
7. Predict genres for unseen movie descriptions
